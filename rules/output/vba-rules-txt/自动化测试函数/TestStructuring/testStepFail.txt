1. 函数名
testStepFail

2. 函数描述
用于报告测试步骤中的失败信息，同时将失败的信息记录在测试报告中。此函数也会改变测试用例的状态为Fail状态（除了此时用例状态为Error）。

3. 函数体
testStepFail(levelOfDetail, 'description')

4. 返回值
无

5. 参数
| 名称          | 类型 | 描述                                                                                     |
|---------------|------|------------------------------------------------------------------------------------------|
| levelOfDetail | str  | 可以用一个数字来表示这个测试步骤有多重要。0表示“非常重要”，数字越大，重要程度越低。没有明确的规范 |
| description   | str  | 表示此测试用例步骤的描述信息，字符串类型，也可以不写，默认是空字符，会写在测试报告中             |

6. 内置函数
无

7. 示例
示例功能：当返回值res等于0时，调用该函数表明该测试步骤失败
示例代码：
@testcase
def checkUndefinedMessage():
    if (res == 0):
        testStepFail(3.0, 'Communication prepared failed')
代码注释：
- @testcase 表示这是一个测试用例
- 当res等于0时，调用testStepFail函数报告测试步骤失败
- 参数3.0表示该步骤的重要性级别
- 'Communication prepared failed'是失败描述信息

8. 其他说明
- 该函数会改变测试用例的状态为Fail状态（除非当前状态已经是Error）
- 失败信息会被记录在测试报告中
