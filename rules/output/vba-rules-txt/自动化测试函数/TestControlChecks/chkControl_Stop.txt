1. 函数名  
chkControl_Stop  

2. 函数描述  
用于停止检测事件  

3. 函数体  
result = chkControl_Stop(object)  

4. 返回值  
名称 | 类型 | 描述  
res | int | 0 成功，-1 失败  

5. 参数  
名称 | 类型 | 描述  
object | object | 检测对象  

6. 内置函数  
无  

7. 示例  
示例功能：停止检测事件  
示例代码：  
```vba
# 创建检测对象，并开始检测错误帧  
checkId = chkStart_ErrorFramesOccured();  
# 停止检测错误帧数量。  
chkControl_Stop(checkId);  
```  
代码注释：  
- `chkStart_ErrorFramesOccured` 用于创建并启动错误帧检测对象。  
- `chkControl_Stop` 用于停止该检测对象的运行。
