7. 示例  
**示例功能**：监控CAN消息0x101的出现次数是否为50次（允许±2次误差）。  

```vba
Sub Example_chkStart_MsgOccurrenceCount()  
    ' 启动消息出现次数检查  
    Dim result As Boolean  
    result = chkStart_MsgOccurrenceCount("0x101", 50, 2, AddressOf CheckCallback)  

    If result Then  
        WriteInfo "消息0x101的计数检查已启动"  
    Else  
        WriteWarningInfo "检查启动失败"  
    End If  
End Sub  

' 回调函数定义  
Sub CheckCallback(actualCount As Integer)  
    If actualCount >= 48 And actualCount <= 52 Then  
        testStepPass "消息0x101出现次数符合预期"  
    Else  
        testStepFail "实际出现次数：" & actualCount  
    End If  
End Sub  
```
