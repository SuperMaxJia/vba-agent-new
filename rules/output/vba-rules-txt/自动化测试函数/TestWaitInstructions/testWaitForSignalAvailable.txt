1. 函数名  
testWaitForSignalAvailable  

2. 函数描述  
用于等待一个指定的信号是否在指定的时间内出现。  

3. 函数体  
```vba
testWaitForSignalAvailable(signal, timeout = 1000)
```  

4. 返回值  
| 名称 | 类型 | 描述 |  
|------|------|------|  
| res  | int  | 0代表超时，1代表事件发生，-1代表出现错误 |  

5. 参数  
| 名称     | 类型   | 描述 |  
|----------|--------|------|  
| signal   | object | 信号对象 |  
| timeout  | int    | 超时的时间，如果设置为0代表会一直阻塞 |  

6. 内置函数  
无  

7. 示例  
**示例功能**：等待一个指定的信号是否在指定的时间内出现在总线上。  
**示例代码**：  
```vba
testWaitForSignalAvailable(signal, timeout = 1000)
```  
**代码注释**：  
- `signal`：需要等待的信号对象。  
- `timeout`：超时时间（毫秒），默认值为1000。
