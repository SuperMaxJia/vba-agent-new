1. 函数名  
testWaitForSignalInRange  

2. 函数描述  
用于等待待检测信号或系统变量是否满足定义的数值范围。  

3. 函数体  
```vba  
testWaitForSignalInRange(object, aLowLimit, aHighLimit, timeout)  
```  

4. 返回值  
| 名称 | 类型 | 描述 |  
|------|------|------|  
| res  | int  | 0代表超时，1代表事件发生，-1发生错误 |  

5. 参数  
| 名称       | 类型   | 描述 |  
|------------|--------|------|  
| object     | object | 信号或者变量对象，系统变量必须是数值类型 |  
| aLowLimit  | int    | 最小值，数值类型 |  
| aHighLimit | int    | 最大值，数值类型 |  
| timeout    | int    | 超时的时间 |  

6. 内置函数  
无  

7. 示例  
**示例功能**：等待待检测信号或系统变量是否满足定义的数值范围。  
**示例代码**：  
```vba  
testWaitForSignalInRange(CAN1.Message.signal, 0, 3, 5000)  
```  
**代码注释**：  
- 程序将阻塞5000ms，如果信号值在0-3范围内，则停止阻塞，继续运行。

无
