1. 函数名
testWaitForSignalsAvailable

2. 函数描述
用于等待一个指定的节点的所有发送信号是否在指定时间内出现

3. 函数体
testWaitForSignalsAvailable(Node, timeout=1000)

4. 返回值
名称 类型 描述
res int 0代表超时，1代表事件发生，-1代表出现错误

5. 参数
名称 类型 描述
Node object Node节点对象
timeout int 超时的时间，如果设置为0代表会一直阻塞

6. 内置函数
无

7. 示例
示例功能：等待一个指定的节点的所有发送信号是否在指定时间内出现
示例代码：
testWaitForSignalsAvailable(Node, timeout=1000)
代码注释：程序将阻塞1000ms，node节点下的信号将停止阻塞，继续运行。

8. 其他和函数或变量强相关的内容
无
