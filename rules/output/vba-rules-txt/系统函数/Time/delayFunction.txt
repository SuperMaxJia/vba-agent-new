1. 函数名
delayFunction

2. 函数描述
指定延时时间后执行函数

3. 函数体
delayFunction(delaytime, mode, func, *parm)

4. 返回值
无

5. 参数
名称        类型    描述
delaytime   int     延时时间，int类型，单位是毫秒
mode        int     0：阻塞模式；1为非阻塞模式
func        function函数名称
*parm       *parm   函数的参数

6. 内置函数
无

7. 示例
示例功能：延时100毫秒后执行函数demoFunction
示例代码：
def demoFunction(m, n):
    writeInfo(m + n)
delayFunction(100, 0, demoFunction, 1, 2)

8. 其他和函数或变量强相关的内容
无
