1. 函数名  
diagGetFlashFileSize  

2. 函数描述  
获取刷写文件的大小  

3. 函数体  
diagGetFlashFileSize(handle)  

4. 返回值  
| 名称 | 类型 | 描述               |  
|------|------|--------------------|  
| size | int  | 文件大小，单位为byte |  

5. 参数  
| 名称   | 类型 | 描述     |  
|--------|------|----------|  
| handle | file | 文件句柄 |  

6. 内置函数  
无  

7. 示例  
功能：获取刷写文件的大小  
代码：  
```vba  
driverFileHandle = diagLoadFlashFile(r'D:\flash_dirve.s19')  
size = diagGetFlashFileSize(driverFileHandle)  
```  
注释：  
- `diagLoadFlashFile` 用于加载刷写文件并返回文件句柄  
- `diagGetFlashFileSize` 通过文件句柄获取文件大小
