1. 函数名
diagLoadFlashFile

2. 函数描述
加载刷写数据文件，支持HEX/S19/MOT解析，正常返回文件句柄

3. 函数体
diagLoadFlashFile(path)

4. 返回值
情况1：正确读值
名称 | 类型 | 描述
file | file | 正常返回文件句柄

情况2：异常情况
名称 | 类型 | 描述
res | int | 返回1，代表文件不存在

情况3：异常情况
名称 | 类型 | 描述
res | int | 返回2，代表文件加载失败

情况4：异常情况
名称 | 类型 | 描述
res | int | 返回3，代表标准文件太大

情况5：异常情况
名称 | 类型 | 描述
res | int | 返回4，代表文件格式不正确

情况6：异常情况
名称 | 类型 | 描述
res | int | 返回5，代表获取文件数据失败

情况7：异常情况
名称 | 类型 | 描述
res | int | 返回6，代表文件关闭失败

5. 参数
名称 | 类型 | 描述
path | String | 加载文件的路径

6. 内置函数
无

7. 示例
示例功能：加载刷写数据文件flash_dirve.s19
示例代码：
driverFileHandle = diagLoadFlashFile(r'D:\flash_dirve.s19')

无
