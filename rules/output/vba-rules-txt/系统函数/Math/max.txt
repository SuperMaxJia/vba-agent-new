1. 函数名  
max  

2. 函数描述  
返回给定参数的最大值，参数可以为序列。  

3. 函数体  
```vba  
max(x, y, z, ....)  
```  

4. 返回值  
| 名称    | 类型 | 描述                  |  
|---------|------|-----------------------|  
| maxNum  | int  | 返回给定参数的最大值  |  

5. 参数  
| 名称 | 类型 | 描述          |  
|------|------|---------------|  
| x    | int  | 数学表达式    |  
| y    | int  | 数学表达式    |  
| z    | int  | 数学表达式    |  

6. 内置函数  
无  

7. 示例  
**示例功能**：求取最大值，返回1000。  
**示例代码**：  
```vba  
max(80, 100, 1000)  
```  
**代码注释**：比较80、100和1000，返回最大值1000。  

8. 其他和函数或变量强相关的内容  
无
