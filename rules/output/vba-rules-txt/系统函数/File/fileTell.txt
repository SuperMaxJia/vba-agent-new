1. 函数名
fileTell

2. 函数描述
返回文件当前指针的位置

3. 函数体
fileTell(file)

4. 返回值
名称 | 类型 | 描述
len | int | 返回文件当前指针的位置

5. 参数
名称 | 类型 | 描述
file | file | 文件对象

6. 内置函数
无

7. 示例
示例功能：获取文件当前指针的位置
示例代码：
file = openFile(r'D:\demo.txt', mode='r', encoding='utf-8')
fileTell(file)
closeFile(file)
代码注释：
- 第一行打开文件demo.txt
- 第二行获取文件指针当前位置
- 第三行关闭文件
