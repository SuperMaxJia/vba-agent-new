1. 函数名
fileReadLines

2. 函数描述
从文件读取的所有行，并返回字符串行列表

3. 函数体
fileReadLines(file)

4. 返回值
名称：list
类型：List
描述：返回字符串行列表

5. 参数
名称  类型  描述
file  file  文件对象

6. 内置函数
无

7. 示例
示例功能：从文件读取所有行
示例代码：
file = openFile(r'D:\demo.txt', mode='r', encoding='utf-8')
fileReadLines(file)
closeFile(file)

代码注释：
- 第一行打开文件demo.txt，模式为读取，编码为utf-8
- 第二行读取文件所有行
- 第三行关闭文件
