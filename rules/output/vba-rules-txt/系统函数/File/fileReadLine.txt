1. 函数名  
fileReadLine  

2. 函数描述  
从文件读取整行，包括换行字符。如果`size`指定一个非负参数，返回指定大小的字节，包括换行字符。  

3. 函数体  
```vba
fileReadLine(file, size = -1)
```  

4. 返回值  
| 名称 | 类型 | 描述 |  
|------|------|------|  
| len  | int  | 返回从字符串中读取的字节 |  

5. 参数  
| 名称 | 类型 | 描述 |  
|------|------|------|  
| file | file | 文件对象 |  
| size | int  | 从文件中读取的字节数 |  

6. 内置函数  
无  

7. 示例  
**示例功能**：从文件读取5个字节  
**示例代码**：  
```vba
file = openFile(r'D:\demo.txt', mode='r', encoding='utf-8')  
fileReadLine(file, 5)  
closeFile(file)  
```  
**代码注释**：  
- `openFile`打开文件`demo.txt`，指定读取模式（`'r'`）和编码（`'utf-8'`）。  
- `fileReadLine`读取文件中的5个字节（包括换行符）。  
- `closeFile`关闭文件对象。  

8. 其他说明  
- 若`size`为-1（默认值），函数返回整行内容。  
- 文件操作需确保路径和权限正确，避免运行时错误。
