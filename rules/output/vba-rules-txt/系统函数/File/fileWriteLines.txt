1. 函数名
fileWriteLines

2. 函数描述
向文件中写入一序列的字符串,字符串序列中需要加入指定的换行符

3. 函数体
fileWriteLines(file, seq)

4. 返回值
名称 | 类型 | 描述
len | int | 返回写入的字符长度

5. 参数
名称 | 类型 | 描述
file | file | 文件对象
seq | String或者()或者List | 字符串序列，可以是元组、列表

6. 内置函数
无

7. 示例
示例功能：向文件中写入指定序列的字符串
示例代码：
file = openFile(r'D:\demo.txt', mode='w', encoding='utf-8')
fileWriteLines(file, '第一行\n', '第二行\n')
closeFile(file)
