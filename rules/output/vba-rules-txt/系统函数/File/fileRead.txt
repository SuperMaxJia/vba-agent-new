1. 函数名  
fileRead  

2. 函数描述  
返回从文件读取指定大小的字节，如果size未给定或为负数则读取所有。  

3. 函数体  
fileRead(file, size = -1)  

4. 返回值  
名称 | 类型 | 描述  
------|------|------  
len | int | 返回从字符串中读取的字节  

5. 参数  
名称 | 类型 | 描述  
------|------|------  
file | file | 文件对象  
size | int | 从文件中读取的字节数  

6. 内置函数  
无  

7. 示例  
示例功能：从文件读取5个字节  
示例代码：  
```vba  
file = openFile(r'D:\demo.txt', mode='r', encoding='utf-8')  
fileRead(file, 5)  
closeFile(file)  
```  
代码注释：  
- `openFile` 用于打开文件，指定路径、模式和编码。  
- `fileRead` 读取指定字节数（此处为5）。  
- `closeFile` 关闭文件。
