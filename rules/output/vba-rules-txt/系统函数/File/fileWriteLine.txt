1. 函数名  
fileWriteLine  

2. 函数描述  
将字符串写入文件，返回的是写入的字符长度。  

3. 函数体  
```vba
fileWriteLine(file, str)
```

4. 返回值  
| 名称 | 类型 | 描述           |  
|------|------|----------------|  
| len  | int  | 返回写入的字符长度 |  

5. 参数  
| 名称 | 类型   | 描述       |  
|------|--------|------------|  
| file | file   | 文件对象   |  
| str  | String | 字符串内容 |  

6. 内置函数  
无  

7. 示例  
**功能**：向文件中写入指定的字符串。  
**示例代码**：  
```vba
file = openFile(r'D:\demo.txt', mode='w', encoding='utf-8')  
fileWriteLine(file, 'd')  
closeFile(file)  
```  
**代码注释**：  
- 使用 `openFile` 打开文件（路径为 `D:\demo.txt`，模式为写入 `'w'`，编码为 `utf-8`）。  
- 调用 `fileWriteLine` 将字符串 `'d'` 写入文件。  
- 使用 `closeFile` 关闭文件对象。  

8. 其他  
无其他与函数或变量强相关的内容。
