1. 函数名
strRight

2. 函数描述
从字符串sourceStr右边截取length长度的字符串

3. 函数体
strLeft(sourceStr, length)

4. 返回值
名称 | 类型 | 描述
str | String | 返回截取的字符串

5. 参数
名称 | 类型 | 描述
sourceStr | String | 字符串
length | int | 截取字符的长度

6. 内置函数
无

7. 示例
示例功能：截取字符串'abcdef'，返回'def'
示例代码：strRight('abcdef', 3)
代码注释：从字符串'abcdef'的右侧截取3个字符，返回结果为'def'
