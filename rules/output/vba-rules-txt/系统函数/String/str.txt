1. 函数名  
str  

2. 函数描述  
将其他类型转换为字符串类型。  

3. 函数体  
```vba
str(object)  
```  

4. 返回值  
| 名称 | 类型   | 描述             |  
|------|--------|------------------|  
| str  | String | 返回字符串类型   |  

5. 参数  
| 名称   | 类型 | 描述                     |  
|--------|------|--------------------------|  
| object | int  | 需要转换为字符串的对象   |  

6. 内置函数  
无  

7. 示例  
**示例功能**：将数值50转换为字符串‘50’。  
**示例代码**：  
```vba
str(50)  
```  
**代码注释**：调用`str`函数将整数50转换为字符串类型。

无
