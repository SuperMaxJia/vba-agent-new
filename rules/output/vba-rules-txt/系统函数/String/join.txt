1. 函数名  
join  

2. 函数描述  
将序列seq中的元素以分隔符code连接成一个字符串。  

3. 函数体  
```vba  
join(seq, code = '')  
```  

4. 返回值  
| 名称     | 类型   | 描述               |  
|----------|--------|--------------------|  
| joinStr  | String | 返回连接好的字符串 |  

5. 参数  
| 名称 | 类型         | 描述                     |  
|------|--------------|--------------------------|  
| seq  | List或者元组 | 序列，可以是列表、元组   |  
| code | char         | 分隔符，默认为空字符串'' |  

6. 内置函数  
无  

7. 示例  
**示例功能**：将元组中的字符串用指定分隔符连接  
**示例代码**：  
```vba  
join(('ds', 'mo'), code='k')  
```  
**代码注释**：  
- 输入元组('ds', 'mo')和分隔符'k'，返回结果为'dskmo'。
