1. 函数名  
strContains  

2. 函数描述  
判断字符串 `targetStr` 是否包含在字符串 `sourceStr` 中。  

3. 函数体  
```vba
strContains(sourceStr, targetStr)
```  

4. 返回值  
| 名称 | 类型    | 描述               |  
|------|---------|--------------------|  
| flag | boolean | 返回布尔类型结果   |  

5. 参数  
| 名称      | 类型   | 描述       |  
|-----------|--------|------------|  
| sourceStr | String | 源字符串   |  
| targetStr | String | 目标字符串 |  

6. 内置函数  
无  

7. 示例  
**示例功能**：检查字符串 `'abcd'` 是否包含子串 `'ab'`。  
**示例代码**：  
```vba
strContains('abcd', 'ab')
```  
**代码注释**：  
- 函数返回 `True`，因为 `'abcd'` 包含 `'ab'`。
