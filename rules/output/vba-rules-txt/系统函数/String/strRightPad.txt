1. 函数名  
strRightPad  

2. 函数描述  
将字符串 `sourceStr` 设置成长度为 `size` 的字符串，如果长度不足则在右侧补充 `code` 字符串。  

3. 函数体  
```vba
strRightPad(sourceStr, size, code)
```

4. 返回值  
| 名称 | 类型   | 描述               |  
|------|--------|--------------------|  
| str  | String | 返回修改后的字符串 |  

5. 参数  
| 名称      | 类型   | 描述                           |  
|-----------|--------|--------------------------------|  
| sourceStr | String | 原始字符串                     |  
| size      | int    | 目标字符串长度                 |  
| code      | char   | 补充字符，默认为空字符串（''） |  

6. 内置函数  
无  

7. 示例  
**示例功能**：将字符串 `'ABCdedf'` 补足到长度为 10，右侧补充字符 `'d'`。  
**示例代码**：  
```vba
strRightPad('ABCdedf', 10, 'd')
```  
**代码注释**：  
- 输入字符串 `'ABCdedf'` 长度为 7，不足 10，右侧补充 3 个 `'d'`。  
- 最终返回字符串 `'ABCdedfddd'`。  

8. 其他  
无其他相关说明。
