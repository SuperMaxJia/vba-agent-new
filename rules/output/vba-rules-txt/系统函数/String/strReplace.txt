1. 函数名  
strReplace  

2. 函数描述  
从字符串 sourceStr 中替换第一次出现的 old 字符串为 new 的新字符串。  

3. 函数体  
```vba  
strReplace(sourceStr, old, new)  
```  

4. 返回值  
| 名称 | 类型   | 描述               |  
|------|--------|--------------------|  
| str  | String | 返回替换后的字符串 |  

5. 参数  
| 名称      | 类型   | 描述             |  
|-----------|--------|------------------|  
| sourceStr | String | 字符串           |  
| old       | String | 被替换的字符串   |  
| new       | String | 替换的字符串     |  

6. 内置函数  
无  

7. 示例  
**示例功能**：替换字符串 'abcdef' 中的 'd' 为 'k'，返回 'abckef'  
**示例代码**：  
```vba  
strReplace('abcdef', 'd', 'k')  
```  
**代码注释**：  
- 输入字符串为 'abcdef'  
- 将 'd' 替换为 'k'  
- 返回结果为 'abckef'
