1. 函数名  
strMid  

2. 函数描述  
从字符串 `sourceStr` 的 `start` 位置开始截取 `length` 长度的子串。  

3. 函数体  
```vba
strMid(sourceStr, start, length)
```  

4. 返回值  

| 名称 | 类型   | 描述           |  
|------|--------|----------------|  
| str  | String | 返回截取的字符串 |  

5. 参数  

| 名称      | 类型   | 描述               |  
|-----------|--------|--------------------|  
| sourceStr | String | 原始字符串         |  
| start     | int    | 起始索引（从0开始） |  
| length    | int    | 截取的字符长度      |  

6. 内置函数  
无  

7. 示例  

**功能**  
从字符串 `'abcdef'` 中截取从索引 `2` 开始的 `3` 个字符，结果为 `'cde'`。  

**示例代码**  
```vba
strMid('abcdef', 2, 3)  
```  

**代码注释**  
- `'abcdef'`：原始字符串。  
- `2`：起始索引（0-based）。  
- `3`：截取长度。  

8. 其他相关说明  
- 索引从 `0` 开始计算。  
- 若 `start` 或 `length` 超出字符串范围，可能返回空字符串或部分截取结果。
