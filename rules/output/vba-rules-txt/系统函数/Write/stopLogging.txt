1. 函数名  
stopLogging  

2. 函数描述  
停止指定的logger模块的录制。当loggerConfig为None时，该方法根据logger_name和file_name可以控制VBA中Loggers下的节点停止录制；当loggerConfig不为None时，则logger_name和file_name两个参数失效，该方法将根据loggerConfig的属性来控制自定义的录制文件停止录制。  

3. 函数体  
```vba  
stopLogging(logger_name, file_name, loggerConfig)  
```  

4. 返回值  
无  

5. 参数  

| 名称         | 类型   | 描述                                                                 |  
|--------------|--------|----------------------------------------------------------------------|  
| logger_name  | String | 工程Loggers模块下配置的logger节点名称                               |  
| file_name    | String | 录制文件的名称                                                     |  
| loggerConfig | String | logger的配置，可以在指定的目录下创建一个录制文件，自定义文件类型（asc, blf, pacp），过滤条件等，实现录制报文的功能；如果目录相同，则会覆盖之前录制文件 |  

6. 内置函数  
无  

7. 示例  
**示例功能**：停止指定的logger模块的录制  
**示例代码**：  
```vba  
stopLogging("logger1")  
```  
**代码注释**：  
- 调用`stopLogging`函数，传入logger节点名称`"logger1"`，停止该logger的录制。
