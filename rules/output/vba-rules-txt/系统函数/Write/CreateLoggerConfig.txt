1. 函数名
CreateLoggerConfig

2. 函数描述
logger的配置,可以在指定的目录下创建一个录制文件，自定义文件类型，过滤条件等，实现录制报文的功能；如果目录相同，则会覆盖之前的录制文件。

3. 函数体
loggerConfig = CreateLoggerConfig(loggerPath, fileName, fileType, size, loggerFilter)

4. 返回值
名称         类型          描述
loggerConfig loggerConfig loggerConfig的对象

5. 参数
名称         类型          描述
loggerPath   String        文件保存的路径，路径需要真实可以创建，且属于文件夹
fileName     String        文件的名称，字符串类型，符合window系统文件的命名规范，不符合报错提示
fileType     String        文件的后缀名称，支持asc、blf、pacp格式文件，支持输入大小写，不符合这三种，报错提示
size         int           文件默认大小，整数类型，单位是MB，当超过指定文件大小时，会自动创建新的文件，后缀+1
loggerFilter loggerFilter  文件记录的过滤器对象，可以配置过滤的条件，可以不设置，不设置默认不过滤

6. 内置函数
无

7. 示例
示例功能：logger的配置
示例代码：
loggerConfig = CreateLoggerConfig(loggerPath, fileName, fileType, size, loggerFilter)
