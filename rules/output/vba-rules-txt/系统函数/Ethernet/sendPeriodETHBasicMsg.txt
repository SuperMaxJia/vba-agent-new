1. 函数名  
sendPeriodETHBasicMsg  

2. 函数描述  
用于周期性原始以太网报文发送  

3. 函数体  
```vba
sendPeriodETHBasicMsg(network, EthernetPacket/IPv4Packet/ARPPacket/UDPPacket/TCPPacket/SOMEIPPacket/DOIPPacket/ETHDBCCommand/ETHDBCPDU, VLAN=['0x88A8:0x0003', '0x8100:0x0002'], cycleTime)
```

4. 返回值  
| 名称   | 类型 | 描述                                                                 |
|--------|------|----------------------------------------------------------------------|
| taskid | int  | 小于0代表发送失败，具体错误类型可参考通讯协议的错误类型定义          |

5. 参数  
| 名称                                                                 | 类型                                                                 | 描述                                                                 |
|----------------------------------------------------------------------|----------------------------------------------------------------------|----------------------------------------------------------------------|
| network                                                              | Network                                                              | 网段对象                                                             |
| EthernetPacket/IPv4Packet/ARPPacket/UDPPacket/TCPPacket/SOMEIPPacket/DOIPPacket/ETHDBCCommand/ETHDBCPDU | EthernetPacket, IPv4Packet, UDPPacket, TCPPacket, SOMEIPPacket, DOIPPacket, ICMPv4Packet, DHCPPacket, ETHDBCCommand, ETHDBCPDU | 原始以太网报文                                                       |
| cycleTime                                                           | int                                                                  | 周期的时间，0代表停发                                                |
| VLAN                                                                 | list                                                                 | VLAN的配置，如果是双VLAN，从左到右输入                               |

6. 内置函数  
无  

7. 示例  
示例功能：周期性发送原始以太网报文  
示例代码：  
```vba
sendPeriodETHBasicMsg(network, EthernetPacket/IPv4Packet/ARPPacket/UDPPacket/TCPPacket/SOMEIPPacket/DOIPPacket/ETHDBCCommand/ETHDBCPDU, VLAN=['0x88A8:0x0003', '0x8100:0x0002'], cycleTime)
```
代码注释：  
- `network`：指定网段对象  
- `EthernetPacket/IPv4Packet/...`：指定要发送的原始以太网报文类型  
- `VLAN`：配置VLAN参数，支持双VLAN配置  
- `cycleTime`：设置发送周期时间，0表示停止发送  

8. 其他相关说明  
- 该函数适用于需要周期性发送原始以太网报文的场景  
- VLAN参数支持灵活配置，适用于单VLAN或双VLAN场景  
- 返回值`taskid`可用于判断发送是否成功，负值表示失败
