1. 函数名
ARPMsg

2. 函数描述
创建一个自定义的ARP报文对象，然后通过设置其对象的属性来设置这条报文相应的参数

3. 函数体
ARPMsg()

4. 返回值
名称      类型      描述
arpMsg    ARPMsg    返回一个报文对象，报文包含以下属性，可分别给下面的属性进行赋值

5. 参数
名称            类型      描述
ether_dhost     int       以太网报文的目标MAC
ether_shost     int       以太网报文的源MAC
arp_op          int       ARP操作字段
arp_sha         string    ARP Mac地址
arp_spa         string    ARP IP地址
arp_dha         string    客户端ARP MAC地址
arp_dpa         string    客户端ARP IP地址
data            byte[]    ARP报文的数据场
address         int       地址信息
op              int       设置ARP操作字段

6. 内置函数
函数名称                    描述
getEtherDHost()            获取目的MAC地址
getEtherSHost()            获取源MAC地址
getARPop()                 获取ARP操作字段,1：ARP请求 2：ARP应答 3：RARP请求 4：RARP应答
getARPSMacAddress()        获取ARP MAC地址
getARPSIPAddress()         获取ARP IP地址
getARPCMacAddress()        获取客户端ARP MAC地址
getARPCIPAddress()         获取客户端ARP IP地址
getARPData()               获取ARP报文的数据场
setEtherDHost(address)     设置目的MAC地址
setEtherSHost(address)     设置源MAC地址
setARPop(op)               设置ARP操作字段,1：ARP请求 2：ARP应答 3：RARP请求 4：RARP应答
setARPSMacAddress(address) 设置ARP MAC地址
setARPSIPAddress(address)  设置ARP IP地址
setARPCMacAddress(address) 设置客户端ARP MAC地址
setARPCIPAddress(address)  设置客户端ARP IP地址

7. 示例
示例功能：创建一条自定义的ARP报文，并将其发到总线上

示例代码：
arpMsg = ARPMsg()
arpMsg.setEtherDHost(0xFFFFFFFFFFFF)  #设置目的Mac地址
arpMsg.setEtherSHost(0x112233445566)  #设置源Mac地址
arpMsg.setARPop(0x0002)  #设置ARP操作字段,1：ARP请求 2：ARP应答 3：RARP请求 4：RARP应答
arpMsg.setARPSMacAddress(0x112233445566)  #设置ARP Mac地址
arpMsg.setARPSIPAddress('192.168.1.0')  #设置ARP IP地址
arpMsg.setARPCMacAddress(0x223344554455)  #设置客户端ARP MAC地址
arpMsg.setARPCIPAddress('192.168.1.10')  #设置客户端ARP IP地址
sendETHMsg(arpMsg.getARPData(), ETH1)
