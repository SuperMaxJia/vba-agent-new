1. 函数名
UDPMessage

2. 函数描述
用于创建一条UDP报文（走协议栈）

3. 函数体
UDPMessage()

4. 返回值
返回一个UDPMessage对象

5. 参数
| 名称                  | 类型  | 描述                          |
|-----------------------|-------|-------------------------------|
| destination_ip_address | str   | 目标IP地址，格式为字符串       |
| source_port           | int   | 源端口号，一个16位的无符号整数 |
| destination_port      | int   | 目标端口号，一个16位的无符号整数 |
| payload               | list  | 以太网报文的数据payload        |

6. 内置函数
| 函数名称               | 函数描述                     |
|------------------------|------------------------------|
| setDestinationIPAddress | 设置目标IP地址               |
| setSourcePort          | 设置源端口号                 |
| setDestinationPort     | 设置目标端口号               |
| setPayload            | 设置报文payload数据          |

7. 示例
示例功能：创建一条UDP报文（走协议栈）

示例代码：
udpMessage = UDPMessage()
udpMessage.setDestinationIPAddress("192.168.2.2")
udpMessage.setSourcePort(7000)
udpMessage.setDestinationPort(8000)
udpMessage.setPayload([])

代码注释：
1. 创建UDPMessage对象
2. 设置目标IP地址为192.168.2.2
3. 设置源端口号为7000
4. 设置目标端口号为8000
5. 设置payload为空列表

8. 其他说明
- 该函数属于Ethernet模块
- 需要配合协议栈使用
- 报文发送需要使用sendETHMsgByStack等发送函数
