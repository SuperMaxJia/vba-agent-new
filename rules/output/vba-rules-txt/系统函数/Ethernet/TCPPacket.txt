1. 函数名
TCPPacket

2. 函数描述
用于创建一条TCP以太网报文

3. 函数体
TCPPacket()

4. 返回值
返回一个TCPPacket对象实例

5. 参数
名称             类型         描述
source_port      int         源端口值，一个16位的无符号整数（0 - 65535）
destination_port int         目的端口值，一个16位的无符号整数（0 - 65535）
internet_packet  IPv4Packet  IPv4Packet实例，表示要封装的IPv4报文
sequence_number  int         序列号值，一个32位的无符号整数
ack_number       int         确认号值，一个32位的无符号整数
payload          list        数据，一个列表
flag             int         标志位值，一个8位的无符号整数（按位定义不同的控制位）
checksum         int         校验和值，一个16位的无符号整数
urgentPointer    int         紧急指针值，一个16位的无符号整数

6. 内置函数
无

7. 示例
示例功能：创建一条IPv4以太网报文
示例代码：
tcpPacket = TCPPacket()
tcpPacket.setSourcePort(7000)
tcpPacket.setDestinationPort(8000)
tcpPacket.setInternetPacket(IPv4Packet)
tcpPacket.setSequenceNumber(1)
tcpPacket.setACKNumber(0)
tcpPacket.setPayload([])
tcpPacket.setDataOffset(20)  // 默认不显示
tcpPacket.setFlag(0x50)      // 默认不显示
tcpPacket.setChecksum(0x7695) // 如果设置为-1,则会自动计算。默认此行不显示，默认值是-1
tcpPacket.setUrgentPointer(0) // 默认不显示

8. 其他
- 该函数属于Ethernet模块
- 需要先创建IPv4Packet对象作为参数
- checksum参数设置为-1时会自动计算校验和
