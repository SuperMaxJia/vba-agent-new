1. 函数名  
ethTCPClientConnect  

2. 函数描述  
客户端建立TCP连接  

3. 函数体  
```vba
socketId = ethTCPClientConnect(network, TCPIPStack, sourPort, destIP, destPort, timeout)
```  

4. 返回值  
| 名称      | 类型 | 描述                                      |  
|-----------|------|-----------------------------------------|  
| socketid  | int  | 如果建立错误返回错误码，-1代表该接口调用时出现错误 |  

5. 参数  
| 名称         | 类型         | 描述              |  
|--------------|--------------|------------------|  
| network      | NetworkI     | 网段信息          |  
| TCPIPStack   | TCPIPStack   | 本机TCP配置信息   |  
| sourPort     | int          | 本机端口号        |  
| destIP       | str          | 目的IP地址        |  
| destPort     | int          | 目的端口号        |  
| timeout      | int          | 超时时间          |  

6. 内置函数  
无  

7. 示例  
**示例功能**：客户端建立TCP连接  
**示例代码**：  
```vba
socketid = ethTCPClientConnet(network, TCPIPStack, sourPort, destIP, destPort, timeout)
```  
**代码注释**：  
- `network`：网段信息  
- `TCPIPStack`：本机TCP配置信息  
- `sourPort`：本机端口号  
- `destIP`：目的IP地址  
- `destPort`：目的端口号  
- `timeout`：超时时间
