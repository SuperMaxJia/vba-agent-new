1. 函数名  
TCPClientSocket  

2. 函数描述  
创建TCP客户端  

3. 函数体  
```vba
TCPClientSocket(ipType=0, ip='127.0.0.0', port=3434)
```  

4. 返回值  
| 名称      | 类型            | 描述         |  
|-----------|-----------------|--------------|  
| tcpClient | TCPClientSocket | 返回TCP客户端 |  

5. 参数  
| 名称   | 类型   | 描述                     |  
|--------|--------|--------------------------|  
| ipType | int    | 0代表IPv4，1代表IPv6      |  
| ip     | String | 服务器IP地址             |  
| port   | int    | 服务器端口               |  

6. 内置函数  
| 函数名称        | 描述                                   |  
|-----------------|----------------------------------------|  
| connect()       | 跟服务器建立TCP连接                    |  
| sendTCPMsg(data)| 向服务器发送TCP报文                    |  
| getExtend()     | 获取报文的扩展帧标识（0标准帧，1扩展帧）|  
| recvTCPMsg(bufsize)| 接收服务器发送的TCP报文（bufsize为字节数）|  
| close()         | 关闭TCP连接                           |  

**参数或返回值说明**  
| 名称    | 类型           | 描述                          |  
|---------|----------------|-------------------------------|  
| data    | List 或 bytes  | TCP报文                       |  
| extend  | int            | 扩展帧标识（0标准帧，1扩展帧） |  
| bufsize | int            | 接收的字节数                  |  

7. 示例  
**功能**：创建TCP客户端并与指定服务器建立连接  
**示例代码**：  
```vba
tcpClient = TCPClientSocket(ipType=0, ip='192.168.7.2', port=65535)  
tcpClient.connect()  
tcpClient.sendTCPMsg(bytes([0x1, 0x2, 0x3, 0x1, 0x2, 0x3, 0x1, 0x2, 0x3, 0x1, 0x2, 0x3, 0x1, 0x2, 0x3, 0x1]))  
```  

8. 其他  
- 需确保服务器IP和端口正确且网络可达。  
- 发送数据需转换为`bytes`或列表格式。  
- 调用`close()`释放资源。
