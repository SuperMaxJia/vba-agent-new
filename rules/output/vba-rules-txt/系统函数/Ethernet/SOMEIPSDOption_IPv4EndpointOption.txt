1. 函数名
SOMEIPSDOption_IPv4EndpointOption

2. 函数描述
创建一条SOMEIPSD的Option对象

3. 函数体
SOMEIPSDOption_IPv4EndpointOption()

4. 返回值
返回一个SOMEIPSD的Option对象

5. 参数
名称            类型    描述
type            str     0x04:组播, 0x14:单播, 0x24:SDEndPoint
ipv4_address    str     IPv4地址字符串，例如 '192.168.2.0'
protocol        int     协议值，6 (TCP) 或 17 (UDP)
port_number     str     端口号的十六进制表示，例如 0x1234

6. 内置函数
无

7. 示例
示例功能：创建一条SOMEIPSD的Option对象

示例代码：
someipSDoption = SOMEIPSDOption_IPv4EndpointOption()
someipSDoption.setType(False)  // 0x04组播 0x14单播 0x24SDEndPoint Option对应类型不一样
someipSDoption.setIPv4Address('192.168.2.0')
someipSDoption.setProtocol(6)  // 17是UDP 6是TCP
someipSDoption.setPortNumber(0x1234)

8. 其他
无其他相关说明
