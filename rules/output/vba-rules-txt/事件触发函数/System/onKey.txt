1. 函数名  
onKey  

2. 函数描述  
按下键盘指定键的时候会触发此事件，按住不松手会持续触发，支持同时定义多个onKey事件（函数名称不能相同）。  

3. 函数体  
```vba  
@onKey('K')  
def onKeyEvent():  
    pass  
```  

4. 返回值  
无返回值。  

5. 参数  
| 名称 | 类型 | 描述 |  
|------|------|------|  
| 按键 | 按键 | 触发事件的按键必须是单个字符，且在‘a-z’或者‘A-Z’之间。 |  

6. 内置函数  
无。  

7. 示例  
功能：当收到按键事件’K’时触发，并将字符串‘收到按键事件’输出到控制台上。  
代码：  
```vba  
@onKey('K')  
def onKeyEvent():  
    writeInfo('按键触发事件已触发！')  
```  
注释：  
- `@onKey('K')` 表示监听键盘按键 'K'。  
- `writeInfo('按键触发事件已触发！')` 在控制台输出提示信息。  

8. 其他相关说明  
- 支持同时定义多个`onKey`事件，但函数名称不能相同。  
- 按键必须是单个字符，且限于字母范围（a-z 或 A-Z）。
