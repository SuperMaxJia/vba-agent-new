1. 函数名
onPreStop

2. 函数描述
点击停止运行（或设备掉线等其他异常情况导致工程停止）按钮，工程正式停止运行前触发此事件，支持同时定义多个onPreStop事件（函数名称不能相同）。

3. 函数体
@onPreStop
def onPreStopEvent():
    pass

4. 返回值
无

5. 参数
无

6. 内置函数
无

7. 示例
示例功能：在工程停止运行前，并将字符串'工程正在停止'输出到控制台上。
示例代码：
@onPreStop
def onPreStopEvent():
    writeInfo('工程正在停止！')

代码注释：
- @onPreStop: 声明这是一个onPreStop事件函数
- writeInfo(): 内置函数，用于向控制台输出信息

8. 其他说明
- 可以同时定义多个onPreStop事件，但函数名称不能相同
- 该事件在工程正式停止运行前触发
