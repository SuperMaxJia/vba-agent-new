1. 函数名  
onStart  

2. 函数描述  
工程运行成功后触发此事件，支持同时定义多个onStart事件（函数名称不能相同）。  

3. 函数体  
```vba  
@onStart  
def onStartEvent():  
    pass  
```  

4. 返回值  
无返回值。  

5. 参数  
无参数。  

6. 内置函数  
无内置函数调用。  

7. 示例  
示例功能：在工程运行成功后，将字符串“工程运行成功”输出到控制台。  
示例代码：  
```vba  
@onStart  
def onStartEvent():  
    writeInfo('工程运行成功！')  
```  
代码注释：  
- `@onStart` 是事件触发装饰器，表示该函数在工程运行成功后自动执行。  
- `writeInfo` 是内置函数，用于向控制台输出信息。  

8. 其他和函数或变量强相关的内容  
- 可以定义多个`onStart`函数，但函数名称不能相同。  
- 该事件适用于工程启动后执行初始化或通知操作。
