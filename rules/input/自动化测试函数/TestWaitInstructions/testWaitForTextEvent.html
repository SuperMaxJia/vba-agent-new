testWaitForTextEvent

VBA使用帮助文档

testWaitForTextEvent

函数描述：用于等待一个来自测试模块的事件文本（只能触发一次）。

函数体

testWaitForTextEvent(text，timeout=0)

输入参数

名称 类型 描述
text str 测试对比的文本
timeout int 超时的时间，如果设置为0代表会一直阻塞

返回值

名称 类型 描述
res int 0代表超时，1代表选择的yes，2代表用户no，-1代表产生未知错误

示例

等待一个字符串为"ErrorFrame occurred"的事件发生，该事件是由testSupplyTextEvent提供。

result = testWaitForTextEvent("ErrorFrame occurred!", 3000);
# handler ‘on errorFrame’ signals event
if (result == 1):
    #事件发生
