##功能##
该函数用于查找目标字符串在源字符串中第一次出现的位置索引，常用于字符串处理和搜索操作，例如在日志分析或数据解析中定位特定子串。

##vba规范##
函数名：strIndexOf
参数：
- sourceStr (String)：源字符串
- targetStr (String)：要查找的目标字符串
返回值：Integer，返回目标字符串首次出现的索引（从0开始），未找到则返回-1

##capl规范##
CAPL中使用strstr函数实现类似功能：
参数：
- char[]：源字符串
- char[]：目标字符串
返回值：dword，返回目标字符串首次出现的指针位置（需要转换为索引），未找到返回0

示例1：
```
char source[] = "hello world";
char target[] = "world";
dword pos = strstr(source, target);
int index = pos ? (pos - (dword)&source) : -1;
// index = 6
```

示例2：
```
char source[] = "abc";
char target[] = "x";
dword pos = strstr(source, target);
int index = pos ? (pos - (dword)&source) : -1;
// index = -1
```

##capl名##
strstr

##转换须知##
1. CAPL的strstr返回指针地址而非直接索引，需要计算偏移量
2. CAPL中未找到返回0，而VBA返回-1
3. CAPL索引计算需要类型转换（dword到int）
4. CAPL字符串是字符数组，VBA是String类型

##转换示例##
CAPL原始代码：
```
char data[] = "CAPL test";
dword result = strstr(data, "test");
int idx = result ? (result - (dword)&data) : -1;
```

转换后的VBA代码：
```
Dim index As Integer
index = strIndexOf("CAPL test", "test")
```

关键转换步骤：
1. 将CAPL的字符数组改为VBA字符串
2. 用strIndexOf替换strstr+指针计算逻辑
3. 直接使用函数返回值，无需额外处理
