##功能##
该函数用于向控制台打印正常消息，主要用于调试或信息输出场景，帮助开发者在程序运行时查看特定信息。

##vba规范##
函数名：writeInfo
参数：info (String类型)，表示需要打印到控制台的信息
返回值：无
功能：将传入的字符串信息打印到控制台

##capl规范##
函数名：write
参数：msg (char[]类型)，表示需要打印到控制台的信息
返回值：无
功能：将传入的字符串信息打印到控制台

示例1：
write("打印正常消息");  // 在控制台输出"打印正常消息"

示例2：
char message[50] = "测试信息";
write(message);  // 在控制台输出"测试信息"

##capl名##
write

##转换须知##
1. 参数类型差异：VBA使用String类型，CAPL使用char[]类型
2. 函数名差异：VBA使用writeInfo，CAPL使用write
3. 输出行为相同：两者都会将信息输出到控制台
4. 在CAPL中需要确保字符串以null结尾

##转换示例##
1. CAPL原始代码：
write("车辆速度: 60km/h");

2. 转换后的VBA代码：
writeInfo "车辆速度: 60km/h"

3. 关键转换步骤说明：
- 将CAPL的write函数名改为VBA的writeInfo
- 移除CAPL语句末尾的分号
- 在VBA中直接使用字符串参数，无需char[]声明
