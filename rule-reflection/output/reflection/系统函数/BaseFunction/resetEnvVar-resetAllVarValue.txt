##功能##
该函数用于将所有环境变量重置为其初始值。如果没有设置初始值，数值型变量会被设为0，字符串型变量会被设为空字符串""。适用于需要快速恢复所有环境变量到默认状态的场景，如测试环境初始化或错误恢复。

##vba规范##
函数名：resetAllVarValue
描述：重置所有环境变量到初始值
参数：无
返回值：无
内置函数：无
调用方式：直接调用resetAllVarValue()
影响范围：所有环境变量
默认值处理：数值型设为0，字符串型设为""

##capl规范##
函数名：resetEnvVar
描述：重置所有环境变量到初始值或默认值
参数：无
返回值：无
示例1：
resetEnvVar(); // 重置所有环境变量
示例2：
on key 'r' {
    resetEnvVar(); // 按键r时重置环境变量
}

##capl名##
resetEnvVar

##转换须知##
1. CAPL中的resetEnvVar与VBA的resetAllVarValue功能完全对应
2. 两者都是无参数无返回值函数
3. 变量重置行为一致：有初始值则恢复初始值，否则数值型为0，字符串型为""
4. CAPL中环境变量作用域与VBA中相同，都是全局范围

##转换示例##
1. CAPL原始代码：
resetEnvVar();

2. 转换后的VBA代码：
resetAllVarValue

3. 关键转换说明：
- 函数名从resetEnvVar改为resetAllVarValue
- 调用语法保持一致，都是直接调用函数名加括号
- 功能和行为完全一致，无需额外修改
