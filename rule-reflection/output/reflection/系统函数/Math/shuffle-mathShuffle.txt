##功能##
该函数用于将序列中的所有元素进行随机排序，常用于需要打乱数据顺序的场景，如洗牌算法、随机抽样等。

##vba规范##
函数名：mathShuffle
参数：seq（List类型，表示需要随机排序的列表）
返回值：无（直接修改原列表）
功能：原地随机打乱列表元素的顺序

##capl规范##
CAPL中没有内置的随机打乱数组的函数，但可以通过自定义函数实现：
1. 使用random函数生成随机索引
2. 通过临时变量交换数组元素
示例1：
```
byte array[3] = {10,7,9};
shuffle(array);
```
示例2：
```
int deck[52];
// 初始化牌组...
shuffle(deck);
```

##capl名##
shuffle

##转换须知##
1. CAPL需要自定义实现shuffle功能，而VBA可直接使用
2. CAPL数组是静态的，VBA列表是动态的
3. CAPL需要显式传递数组长度参数
4. CAPL的random函数范围需要特别注意

##转换示例##
1. CAPL原始代码：
```
byte cards[5] = {1,2,3,4,5};
shuffle(cards);
```

2. 转换后的VBA代码：
```vba
Dim cards As Variant
cards = Array(1,2,3,4,5)
mathShuffle cards
```

3. 关键转换说明：
- CAPL的静态数组转换为VBA的动态数组
- 移除数组长度参数
- 函数调用语法从CAPL的括号形式改为VBA的直接参数形式
