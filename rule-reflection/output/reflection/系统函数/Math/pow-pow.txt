##功能##
计算x的y次方的值，用于数学运算中的幂运算。在实际应用中常用于计算面积、体积、指数增长等场景。

##vba规范##
函数名：pow
参数：x(int类型，数学表达式)，y(int类型，数学表达式)
返回值：long类型，返回x的y次方的值
内置函数：无

##capl规范##
函数名：pow
参数：x(long类型)，y(long类型)
返回值：long类型，返回x的y次方的值
示例1：pow(100, 2) // 返回10000
示例2：pow(2, 10) // 返回1024

##capl名##
pow

##转换须知##
1. 参数类型：VBA使用int类型，CAPL使用long类型
2. 返回值类型：VBA返回long类型，CAPL也返回long类型
3. 函数行为：两者功能完全一致
4. 特殊情况：两者都支持负指数运算

##转换示例##
1. CAPL原始代码：
long result;
result = pow(5, 3);

2. 转换后的VBA代码：
Dim result As Long
result = pow(5, 3)

3. 关键转换步骤说明：
- 变量声明从CAPL的long改为VBA的Long
- 函数调用语法保持不变
- 返回值处理方式相同
