##功能##
该函数用于在测试报告中添加外部应用链接（如URL），允许用户指定链接类型、标题和实际路径。主要用于关联需求管理工具（如DOORS）或其他外部资源。

##vba规范##
函数定义：`testReportAddExternalRef(type, title, ref)`
参数：
- type: 字符串类型，指定链接类型（如"url"、"doors"等）
- title: 字符串类型，链接的描述文本
- ref: 字符串类型，链接的实际路径
返回值：无

##capl规范##
函数定义：`testReportAddLink(type, title, ref)`
参数：
- type: char[]，链接类型标识符
- title: char[]，链接显示名称
- ref: char[]，链接地址
返回值：void
示例1：
```capl
testReportAddLink("url", "Specification", "http://example.com/spec");
```
示例2：
```capl
testReportAddLink("doors", "Requirement", "doors://server/path");
```

##capl名##
testReportAddLink

##转换须知##
1. CAPL使用char数组而非VBA的String类型处理字符串
2. 两者都是无返回值(void)函数
3. CAPL函数名与VBA存在差异但功能相同
4. 链接类型参数在两种语言中都需要显式传递

##转换示例##
1. CAPL原始代码：
```capl
testReportAddLink("easee", "Test Case", "easee://testcase/123");
```
2. 转换后的VBA代码：
```vba
testReportAddExternalRef "easee", "Test Case", "easee://testcase/123"
```
3. 关键转换说明：
- 函数名从testReportAddLink改为testReportAddExternalRef
- 参数类型从char[]改为String
- 语法结构从C风格改为VBA风格
